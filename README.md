# AnimoFound 🔍

A Lost and Found Item Management System for **De La Salle University**.

---

## 📖 Overview

AnimoFound is a Java-based system designed to help DLSU students, faculty, and staff report, search for, and reclaim lost items within the university campus. It provides a centralized platform to streamline the lost and found process, reducing the hassle of locating misplaced belongings.

---

## ✨ Features

- 📋 **Report Lost Items** – Submit details about a lost item including description, location last seen, and date.
- 📦 **Report Found Items** – Log found items to help reunite them with their owners.
- 🔎 **Search & Browse** – Search through listed lost and found items to find a match.
- 🧾 **Item Management** – Track the status of items (lost, found, claimed).
- 👤 **User Accounts** – Register and log in to manage your own reports.

## 🚀 Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- A Java IDE (e.g., IntelliJ IDEA, Eclipse, NetBeans) or the command line

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/AnimoFound.git
   cd AnimoFound
   ```

2. **Compile the project**
   ```bash
   javac -d bin src/**/*.java
   ```

3. **Run the application**
   ```bash
   java -cp bin Main
   ```
